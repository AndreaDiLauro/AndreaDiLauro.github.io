# Resume Directory

Place downloadable CV/resume files in this directory.

## Required Files

### PDF Resume
- `andrea-di-lauro-resume.pdf` (current CV in PDF format)
- `andrea-di-lauro-resume-en.pdf` (English version if different)
- `andrea-di-lauro-resume-it.pdf` (Italian version if different)

## File Specifications

- **Format:** PDF
- **Size:** Optimised for web download (under 2MB)
- **Content:** Professional CV with credits, contact info, headshot
- **Filename:** Use consistent naming convention

## Usage

These files are used for:
- Download links in hero section
- Contact page download options
- Casting director submissions
- Agent/manager sharing

## Updating Resume

When updating your CV:
1. Replace the PDF file with the same filename
2. Ensure contact details match the website
3. Keep file size optimised for quick download
4. Test download links after updating
