# Headshots Directory

Place headshot images in this directory following the naming convention:

## Required Files

### Hero Headshot (for homepage)
- `headshot-01-hero.jpg` (2000px wide, 4:5 crop recommended)
- `headshot-01-hero.webp` (WebP version for faster loading)

### Thumbnail Headshots
- `headshot-01-thumb.jpg` (400px wide)
- `headshot-01-thumb.webp` (WebP version)

### Additional Headshots
- `headshot-02-hero.jpg`, `headshot-02-hero.webp`
- `headshot-03-hero.jpg`, `headshot-03-hero.webp`
- etc.

## Image Specifications

- **Format:** JPEG and WebP
- **Hero images:** Max 2000px wide, 4:5 aspect ratio recommended
- **Thumbnails:** 400px wide
- **Quality:** High quality, optimised for web
- **Alt text:** Update in HTML with descriptive captions

## Usage

These images are used in:
- Homepage hero section
- Gallery section
- Social media meta tags
- Press materials
