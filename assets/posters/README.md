# Posters Directory

Place video poster images and promotional materials in this directory.

## Required Files

### Video Posters
- `showreel-poster.jpg` (1280x720, 16:9 aspect ratio)
- `showreel-poster.webp` (WebP version for faster loading)

### Production Posters (Optional)
- `cibele-poster.jpg`
- `girls-poster.jpg`
- `euforia-poster.jpg`

## Image Specifications

- **Format:** JPEG and WebP
- **Size:** 1280x720 for video posters
- **Aspect Ratio:** 16:9 for video content
- **Quality:** High quality, web-optimised

## Usage

These images are used for:
- Video player poster frames
- Social media sharing
- Production showcases
- Press materials
