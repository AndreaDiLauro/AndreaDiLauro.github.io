# Gallery Directory

Place gallery images (headshots, production stills, behind-the-scenes) in this directory.

## Required Files

### Gallery Images (multiple sizes for responsive loading)
- `gallery-01-400.jpg` / `gallery-01-400.webp` (small screens)
- `gallery-01-800.jpg` / `gallery-01-800.webp` (medium screens)
- `gallery-01-1200.jpg` / `gallery-01-1200.webp` (large screens)
- `gallery-01-large.jpg` (full-size for lightbox)

### Headshot Gallery Items
- `headshot-01-400.jpg` / `headshot-01-400.webp`
- `headshot-01-800.jpg` / `headshot-01-800.webp`
- `headshot-01-1200.jpg` / `headshot-01-1200.webp`
- `headshot-01-large.jpg`

## Image Specifications

- **Format:** JPEG and WebP (provide both)
- **Sizes:** 400px, 800px, 1200px widths + large version
- **Quality:** Optimised for web, maintain professional standard
- **Alt text:** Update in HTML with descriptive captions

## Naming Convention

- `gallery-XX-SIZE.format` for production stills
- `headshot-XX-SIZE.format` for portrait shots
- `behind-XX-SIZE.format` for behind-the-scenes

## Usage

These images are used in:
- Gallery section with responsive srcset
- Lightbox modal for full-size viewing
- Social media sharing
